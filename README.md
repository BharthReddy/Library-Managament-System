# Library Management System

## Description
A web-based Library Management System developed using Django that helps manage books, members, book issuing, returning, and history.

## Features
- Admin login
- Add books
- Update books
- Delete books
- View members
- Issue books
- Return books
- Issue history

## Technologies Used
- Python
- Django
- HTML
- CSS
- MySQL

## How to Run
1. Clone the repository
2. Install Django
3. Run:
   python manage.py runserver
